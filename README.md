# dbcli

Просмотр таблиц БД в консоли

Эта универсальная утилита может работать с различными СУБД: Oracle Database, SQLite, PostgreSQL, MySQL, MariaDB, Microsoft SQL Server и другими. В основе своей она обращается к JDBC-драйверу для выбранной СУБД. Необходимые драйверы поставляются в комплекте с программой.

Конфигурация подключения к серверу СУБД хранится в файле "settings.ini".

На вход программа получает наименование семейства СУБД и SQL-запрос типа "select". На выходе она формирует ответ от сервера СУБД в консоль: в частности, таблицу из базы данных по соответствующему SQL-запросу.

Программу можно запустить с ключом "--help", по которому она выдаст подробную справку о себе.

Примеры запуска программы:

```
dbcli.bat --dbtype "sqlserver" --sql "select @@version as v"
```

```
dbcli.bat --dbtype "mariadb" --sql "select version() as v"
```

```
dbcli.bat --dbtype "JDBCExcel" --sql "select TrackId, Name from Track limit 15"
```

Программа является полностью кроссплатформенной: она может работать под Microsoft Windows, Linux, macOS и под любыми другими операционными системами с поддержкой Java, ver. 17.x или новее.
